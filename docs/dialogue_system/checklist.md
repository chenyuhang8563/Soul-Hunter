# Dialogue System Checklist

## UI 与表现 (UI & Presentation)

- [ ] 对话框是否能正确显示在屏幕下方，且不遮挡重要游戏画面？
- [ ] 头像是否能根据当前说话角色动态更新？
- [ ] 角色名称是否显示正确？
- [ ] 对话文本是否支持富文本并且有顺滑的打字机播放效果？
- [ ] 右下角的向下箭头是否在文本播放完毕后才显示，并有明显的浮动提示？

## 交互控制 (Interaction & Controls)

- [ ] 按下 `Space` 键或点击对话框本体时，是否能加速/立即完成当前正在播放的文本？
- [ ] 文本完全显示且无选项时，按下 `Space` 键或点击对话框是否能顺利进入下一句对话？
- [ ] 选项框是否准确出现在对话框的右上角区域？
- [ ] 点击选项按钮是否能跳转到正确的后续对话节点？
- [ ] 按下键盘的 `1`、`2` 等数字键是否能快捷触发对应的选项？
- [ ] 出现选项时，是否屏蔽了“点击/空格进入下一句”的默认逻辑，强制玩家必须做出选择？

## 架构与健壮性 (Architecture & Robustness)

- [ ] 对话系统是否能在任意场景中通过 `DialogueManager` 单例随时调用？
- [ ] 对话结束时，UI 是否正确隐藏并重置内部所有的状态（文本、选项、头像）？
- [ ] 对话结束时是否抛出了事件/信号供其他系统（如任务系统、剧情系统、角色控制系统）监听并恢复游戏状态？
